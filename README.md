# ResumeIQ Pro

ResumeIQ Pro is a modern ATS Resume Analyzer and Career Intelligence Platform designed to help students, job seekers, and professionals improve their resumes and understand their career readiness.

## Features

* ATS Resume Score Analysis
* Resume Strength Evaluation
* Skills Detection
* Missing Skills Analysis
* Career Recommendation Engine
* Job Match Analysis
* Recruiter View Mode
* Improvement Suggestions
* Achievement System
* PDF Report Download
* Responsive Design
* Dark & Light Theme Support

## Live Demo

https://resumeiq-pro-orcin.vercel.app

## Tech Stack

* React
* TypeScript
* TanStack Start
* Tailwind CSS
* Framer Motion
* Vercel

## How It Works

1. Upload your Resume (PDF/DOCX)
2. Analyze ATS Compatibility
3. View Resume Strength Report
4. Discover Suitable Career Paths
5. Identify Missing Skills
6. Download Detailed Report
